igolden dotfiles
===

My freshly polish dotfiles for 2017.


### Usage

```
git clone igolden/dotfiles ~/dotfiles
cd ~/dotfiles
bin/help
```


### Commands

Commands


```bash

bin/help - help screen

bin/dependencies - view system status

bin/install - install dotfiles

bin/uninstall - uninstall dotfiles
```
